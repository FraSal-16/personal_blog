# Personal blog
Personal Blog creato interamente con python e django

# Funzionalità
Piccolo blog personale per gestire post di qualsiasi genere sulle orme di facebook.
Nello specifico: 
- si posso creare post
- la modifica e l'eliminazione dei post è riservata all'admin django e al creatore del post necessariamente registrato al blog
- funzione di login e logout
- rendirizzamenti post login/logout
- finalizzazione estetica (a mio piacere) tramite css html e bootstrap
